FNV Hashing

When adding resources to packages, you're going to need a hash to use for its instance ID. That's what @s4tk/hashing is for.

Functions

The two functions you're going to need the most are fnv32(text: string) and fnv64(text: string). These produce 32-bit and 64-bit FNV hashes, respectively.

If you'd like to force the high bit to 1, you can do so by passing true as the second argument.
