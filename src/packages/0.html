Package

When dealing with packages, you'll need to use the Package class from @s4tk/models.

This tutorial assumes that you are somewhat familiar with MappedModels. If you are not, you should read the Intro to S4TK tutorial before this one.

Creating Package

To create a new, empty package, you can use new Package().

If you want to create a package with existing entries, you can pass a list of key/resource pairs as the first argument to the constructor. Check the docs if you want more information.

Tip: Never use the name "package" for a variable, as package is a reserved word in JavaScript - I usually stick with "pkg".
