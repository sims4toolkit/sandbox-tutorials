Challenge

Now that you've read about the Package class, let's put your skills to the test.

Importing from S4TK

In order to use Package, you need to import it. So... do that. Do it now. Hurry. Chop chop.

Create a Package

You'll need a package to add resources to, so make an empty one and assign it to a variable called pkg.

Create a Resource Key

Let's create a resource key for an English string table. The instance should be the FNV 64 hash of "MyStringTable".

Hint: You'll have to import the hashing function and two enums. Don't forget to set the locale code in the instance.

Import a STBL

This isn't a challenge for STBLs, so don't bother making one from scratch. You can use StringTableResource.from(await Sandbox.import("English.stbl")) to load a stbl into a variable called stbl.

Add STBL to the PKG

Add stbl to your package using resourceKey as its key.

Get the Buffer

Save the package's buffer to a variable called pkgBuffer. This step is important, because it's what your code will be tested against.
