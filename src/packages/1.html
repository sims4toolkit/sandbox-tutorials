Package Binaries

To load an existing package into a Package, or to save one as a file, you need to work with binary DBPF buffers.

"DBPF" stands for "Database Packed File", and it is the name of the binary format that package files use. You don't really need to know this, but I'm telling you anyways ✨.

Reading a DBPF Buffer

First, you have to get a buffer that contains binary package data, and then load that buffer into a Package with from(buffer).

This tutorial provides a package named "Sample.package" - you can import it with await Sandbox.import().

Writing a DBPF Buffer

To save a Package object as a file, you have to get its buffer with the getBuffer() method.

In the sandbox, you can use Sandbox.download(filename, buffer) to save the package to your computer.

Note about Node

When using S4TK in Node, remember that import() and download() are not available. You'll have to use the fs Node module to read & write files.
