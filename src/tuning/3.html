Now that you've read about the XmlResource class, let's put your skills to the test.

Importing from S4TK

In order to use XmlResource, you're going to need to import it. Do that first.

Create New Tuning

Use the constructor to create an XmlResource for the starting XML. Save it to a variable called tuning.

Update the DOM

First, delete the YOUNGADULT enum node from ages*. Then, change the inner value of trait_type to GAMEPLAY.

There's nothing too fancy here, just remember what you have to do to ensure that the content is kept in sync after you edit the DOM.

* Recall from the XML DOMs tutorial that there is currently no built-in way to delete a node. For help, you can review this StackOverflow post.
