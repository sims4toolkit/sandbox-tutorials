Reading and Writing XML

An important aspect of XmlResources is reading from and writing to XML files.

Reading XML

Like any other resource, you can load existing data into an XmlResource model with its static from(buffer: Buffer) method. It accepts a buffer containing plain text XML data, and will load that data into its DOM.

The XmlResource.from() method also accepts some optional arguments, but you usually do not have to worry about these. You can check out the documentation linked above for more information if you wish.
