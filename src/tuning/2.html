Writing to Files

Sometimes, you may want to write an XmlResource's content to an XML file, such as when you're extracting tuning from the game.

Getting the Buffer

Like every other resource, you can get an XmlResource's buffer with getBuffer(). Once you have the buffer, you can write it to a file using whatever your environment allows.

In Node, you'd use the fs module to write your buffer to a file, rather than Sandbox.download().
