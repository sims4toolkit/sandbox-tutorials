When dealing with string tables, you'll need to use the StringTableResource class from @s4tk/models.

CREATING

To create a new, empty string table, you can use new StringTableResource().

Alternatively, if you have a buffer that contains binary STBL data, you can use the StringTableResource.from(buffer) method.

EDITING

The addAndHash() method will add the given string to the table, and use its 32-bit hash as its key.
